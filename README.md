# Date & Age Calculator

A Python-based console application that collects personal birth-date information, validates user input, calculates ages, and identifies the oldest and youngest individuals in the dataset.

## Features

* Collects names and birth dates from multiple users.
* Validates entered dates and filters out invalid records.
* Calculates each person's age based on the current date.
* Sorts people from oldest to youngest.
* Displays the day of the week each person was born.
* Identifies the oldest and youngest person.
* Saves results to a text file and reads them back for display.
* Demonstrates the use of Python dictionaries, functions, loops, exception handling, file I/O, and the `datetime` module.

## Technologies Used

* Python 3
* `datetime` module

## Learning Objectives

This project was developed to practice:

* Data collection and validation
* Dictionary-based data storage
* Age calculation using dates
* Sorting and data processing
* File handling (read/write operations)
* Exception handling in Python

## Output

The program generates a text file (`data.txt`) containing the calculated ages of all valid users and displays the results in the console.
